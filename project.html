Alert-Drive-Dashboard

AlertDrive Dashboard is a driver fatigue monitoring system that detects eye blinking, yawning, and drowsiness using AI and camera-based analysis. It provides real-time alerts and live status updates to help prevent accidents and improve road safety.

Air Piano

Air Piano is a virtual piano system that lets users play music by moving their fingers in the air using camera-based hand tracking and computer vision. It converts gestures into musical notes in real time.

PC Auto Lock System

PC Auto Lock System is a software solution that automatically locks a computer screen when the user is away from their desk. It uses motion sensors and AI to detect inactivity and secure the system.

Virtual Mouse

Virtual Mouse is a computer vision-based system that allows users to control their mouse pointer using hand gestures captured by a camera. It enables intuitive and hands-free navigation of the desktop environment.
